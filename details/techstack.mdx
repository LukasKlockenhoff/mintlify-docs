## Technologies used in Piguesso

Piguesso is a project that uses a variety of technologies to create a fun and interactive game. The game is built with a combination of web technologies and machine learning. The game is built with the following technologies:

<CardGroup cols={2}>
  <Card title="Nextjs" icon="triangle" href="https://www.nextjs.org">
    We use Nextjs to create the frontend of the game because of its performance and ease of use
  </Card>
  <Card title="react" icon="react" href="https://www.react.dev">
    React provides interactivity on the client side
  </Card>
  <Card title="rust" icon="rust" href="https://foundation.rust-lang.org/">
    We use Rust to implement the API to manage the game state because of its performance and safety
  </Card>
